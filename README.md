**[calcular-h5ox.onrender.com](https://calcular-h5ox.onrender.com/)**

## Funcionalidades

### **Quantos dias faltam para...**
- Descubra quantos dias faltam para feriados como Natal, Páscoa, Carnaval
- Calcule dias para uma data personalizada qualquer

### **Bhaskara**
- Resolva equações de segundo grau completas
- Veja o valor de Δ e as raízes x₁ e x₂

### **Delta (Δ)**
- Calcule apenas o valor do discriminante
- Saiba se as raízes são reais, iguais ou complexas

### **Regra de Três**
- Resolva problemas de proporção de forma simples
- Ideal para porcentagens e cálculos proporcionais

### **Fatorial**
- Calcule o fatorial de números de 0 a 20
- Veja o passo a passo do cálculo

### **Em breve...**
- Média, Mediana e Moda
- Teorema de Pitágoras
- Conversor de Moedas
- Conversor de Unidades

## **Sistema de Usuários**
- Login opcional - use como anônimo se preferir
- Seu nome aparece no cabeçalho quando logado

## **Tecnologias Utilizadas**

- **Backend**: Java 17, Spring Boot 4.0.3
- **Frontend**: HTML5, CSS3, Thymeleaf
- **Banco de Dados**: PostgreSQL
- **Build**: Maven
- **Deploy**: Render.com (com Docker)

## **Como Usar**

1. Acesse [calcular-h5ox.onrender.com](https://calcular-h5ox.onrender.com)
2. Navegue pelos cards na página inicial
3. Escolha a calculadora desejada
4. Insira os valores e clique em "Calcular"
5. O resultado aparece na mesma página, sem recarregar!
